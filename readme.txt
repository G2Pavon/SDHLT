====================
---- How to use ----
====================
1. Open Hammer or Batch Compiler's configuration dialog.
2. Set CSG, BSP, VIS, RAD/LIGHT tool paths to 'sdHLCSG.exe', 'sdHLBSP.exe', 'sdHLVIS.exe', 'sdHLRAD.exe' in 'tools' folder.
3. Add 'sdhlt.wad' into your wad list. (null, hint, skip)
4. Add 'sdhlt.fgd' into your fgd list (func_detail, func_group)

If you are running 64-bit Windows, use the 'sdHLCSG_x64.exe', 'sdHLBSP_x64.exe', 'sdHLVIS_x64.exe' and 'sdHLRAD.exe_x64.exe'.
The main benefit of the 64-bit version is no memory allocation failures, because the 64-bit tools have access to more than 2GB of system memory.
